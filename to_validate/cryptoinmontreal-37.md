Write and compile your first lines of code on Remix. Basically, start learning to code on the Blockchain!

For the 37th edition of Crypto in Montreal, Kate Stapleton will introduce the Ethereum platform and show you how to write and compile your first lines of code.

PLEASE BRING your laptop :) Because, you know... it sucks to ask someone to code on their machine while they use it LOL.

This presentation is intended for beginners, with no programming experience necessary.

### Agenda

Kate will break down Smart Contracts, Solidity and give you all the tools you need to get started in Blockchain development. She will cover:

- Introduction to Ethereum: what it is and why we use it
- Breaking down Smart Contracts: what they are and how they work
- Introduction to Solidity: Ethereum's own coding language
- Get to know Remix: how to navigate the platform
- Write and compile your first lines of code on Remix
